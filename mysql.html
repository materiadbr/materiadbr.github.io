MYSQL


Comando CREATE DATABASE

El comando CREATE DATABASE en MySQL es usado para crear una nueva base de datos.

  • Ejemplo: 
SQL

CREATE DATABASE sistema_ventas;

Comando CREATE TABLE

El comando CREATE TABLE en MySQL es usado para crear una nueva tabla en una base de datos existente.

  • Ejemplo: 
SQL

CREATE TABLE clientes (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nombre VARCHAR(255) NOT NULL,
    email VARCHAR(255) UNIQUE
);

Comando CREATE INDEX

El comando CREATE INDEX en MySQL es usado para crear un índice en una o más columnas de una tabla para mejorar el rendimiento de las búsquedas.

  • Ejemplo: 
SQL

CREATE INDEX idx_nombre_cliente ON clientes (nombre);

Comando CREATE TRIGGER

El comando CREATE TRIGGER en MySQL es usado para crear un disparador, un conjunto de instrucciones que se ejecuta automáticamente cuando ocurre un evento (como INSERT, UPDATE o DELETE) en una tabla específica.

  • Ejemplo: 
SQL

CREATE TRIGGER actualizar_stock_despues_de_venta
AFTER INSERT ON pedidos
FOR EACH ROW
UPDATE productos SET stock = stock - NEW.cantidad WHERE id = NEW.producto_id;

Comando CREATE PROCEDURE

El comando CREATE PROCEDURE en MySQL es usado para crear rutinas almacenadas (bloques de código reutilizable) que se ejecutan en el servidor.

  • Ejemplo: 
SQL

CREATE PROCEDURE ObtenerTodosClientes()
BEGIN
  SELECT * FROM clientes;
END;

Comando CREATE EVENT

El comando CREATE EVENT en MySQL es usado para crear un evento, una tarea programada que se ejecuta automáticamente en un horario específico.

  • Ejemplo: 
SQL

CREATE EVENT limpiar_registrzos
ON SCHEDULE AT CURRENT_TIMESTAMP + INTERVAL 1 MONTH
DO
  DELETE FROM registros WHERE fecha < NOW() - INTERVAL 1 YEAR;

Comando ALTER TABLE

El comando ALTER TABLE en MySQL es usado para modificar la estructura de una tabla. Puedes agregar, eliminar o modificar columnas, y cambiar restricciones.

  • Ejemplo: 
SQL

ALTER TABLE clientes
ADD COLUMN telefono VARCHAR(20),  -- Añadir una columna
MODIFY COLUMN email VARCHAR(255) NOT NULL; -- Modificar una columna

Comando RENAME TABLE

El comando RENAME TABLE en MySQL es usado para cambiar el nombre de una tabla.

  • Ejemplo: 
SQL

RENAME TABLE clientes TO clientes_registrados;

Comando DROP

El comando RENAME TABLE en MySQL es usado para eliminar completamente un objeto de la base de datos (tablas, bases de datos, índices, etc.).

  • Ejemplo: 
SQL

DROP TABLE clientes;
DROP VIEW clientes_importantes;

Comando TRUNCATE TABLE

El comando RENAME TABLE en MySQL es usado para eliminar todas las filas de una tabla, reiniciando su AUTO_INCREMENT. Es mucho más rápido que DELETE para eliminar todos los registros.

  • Ejemplo: 
SQL

TRUNCATE TABLE clientes;

Comando CREATE USER

El comando CREATE USER en MySQL es usado para crear una nueva cuenta de usuario en el servidor.

  • Ejemplo: 
SQL

CREATE USER 'administrador'@'localhost' IDENTIFIED BY 'contraseña_segura';

Comando GRANT

El comando GRANT en MySQL es usado para conceder permisos a un usuario para realizar acciones específicas en la base de datos.

  • Ejemplo: 
SQL

GRANT ALL PRIVILEGES ON sistema_ventas.* TO 'administrador'@'localhost';

Comando REVOKE

El comando REVOKE en MySQL es usado para retirar permisos previamente concedidos a un usuario.

  • Ejemplo: 
SQL

REVOKE DELETE ON sistema_ventas.* FROM 'administrador'@'localhost';
