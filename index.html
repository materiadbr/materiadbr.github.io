BASE DE DATOS

Una base de datos es una colección organizada de datos que se almacena y gestiona de forma electrónica. Piensa en ella como un archivo o un conjunto de tablas interrelacionadas, similar a una hoja de cálculo, pero mucho más potente y estructurada.

El objetivo principal de una base de datos es permitir que la información se almacene, se recupere, se modifique y se gestione de manera eficiente y segura.

ESTRUCTURA DE BASE DE DATOS

TABLA

Una tabla es el componente principal de una base de datos. Piénsala como una hoja de cálculo completa que contiene información sobre una entidad específica, como "Clientes", "Productos" o "Empleados".

  • Función: Organiza los datos de manera lógica y coherente.
  • Ejemplo: En una base de datos de una tienda, podrías tener una tabla llamada PRODUCTOS que almacene la información de cada artículo en venta.

ATRIBUTO (COLUMNA)

Los atributos son las características o propiedades que describen a la entidad de la tabla. Son las columnas de la tabla.

  • Función: Definen el tipo de dato que se almacenará en cada columna.
  • Ejemplo: En la tabla PRODUCTOS, los atributos serían ID_Producto, Nombre, Precio y Stock.

TUPLA (FILA)

Una tupla es una instancia individual de la entidad, es decir, una fila completa en la tabla. Cada registro contiene un valor para cada uno de los atributos.

  • Función: Representan una entrada única de información.
  • Ejemplo: En la tabla PRODUCTOS, un registro sería:

  • ID_Producto: 101
  • Nombre: Camiseta de algodón
  • Precio: 25.50
  • Stock: 150
